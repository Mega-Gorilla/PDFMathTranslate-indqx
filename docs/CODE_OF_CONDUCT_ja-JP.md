# コントリビューター行動規範

## 私たちの誓約

私たちメンバー、コントリビューター、およびリーダーは、年齢、体格、目に見えるまたは目に見えない障がい、民族、性的特徴、ジェンダー・アイデンティティおよび表現、経験年数、学歴、社会経済的地位、国籍、容姿、人種、宗教、性的指向にかかわらず、すべての人にとって嫌がらせのないコミュニティを実現することを誓います。

私たちは、オープンで歓迎的、多様で包摂的、健全なコミュニティを育む行いと言動を取ることを誓います。

## 私たちの標準

前向きな環境づくりに寄与する行動の例:

- 他者に対する思いやりと親切心を示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを提供し、真摯に受け止めること
- 誤りを犯した場合は影響を受けた人に謝罪し、経験から学ぶこと
- 個人の利益だけでなくコミュニティ全体にとって最善を考えること

容認できない行動の例:

- 性的な言動や画像、またはあらゆる種類の性的関心やアプローチ
- 荒らし行為、侮辱的・差別的なコメント、個人攻撃や政治的攻撃
- 公私を問わない嫌がらせ
- 明示的な許可なく他者の住所やメールアドレスなどの個人情報を公開すること
- 職業的な場で不適切と合理的に判断されるその他の行為

## 執行責任

コミュニティのリーダーは、受け入れ可能な行動の基準を明確にし施行する責任を負います。リーダーは、不適切・脅迫的・攻撃的・有害と判断した行為に対して、公平で適切な是正措置を講じます。

リーダーは、この行動規範に沿わないコメント、コミット、コード、Wiki 編集、Issue などの投稿を削除・編集・却下する権限と責任を持ち、必要に応じてモデレーションの理由を説明します。

## 適用範囲

本行動規範は、コミュニティ内の全ての場所に適用され、コミュニティを公の場で公式に代表する個人にも適用されます。公式メールアドレスの使用、公認のソーシャルメディアアカウントからの投稿、オンライン／オフラインイベントでの代表などが該当します。

## 執行

暴言、嫌がらせ、その他受け入れがたい行動を目撃または経験した場合は、執行を担当するコミュニティリーダー（aw@funstory.ai）まで報告してください。すべての苦情は迅速かつ公平に確認・調査されます。

コミュニティリーダーは、通報者のプライバシーと安全を尊重する義務があります。

## 執行ガイドライン

コミュニティリーダーは以下のコミュニティ影響ガイドラインに従い、この行動規範違反があった場合の結果を判断します。

### 1. 是正

**コミュニティへの影響**: 不適切な言葉の使用や、職業的・歓迎的でないとみなされる行動。

**結果**: コミュニティリーダーからの非公開の書面による警告。違反の内容とその不適切さを明確にし、必要に応じて公開謝罪を求める場合があります。

### 2. 警告

**コミュニティへの影響**: 単一または複数回の違反。

**結果**: 今後同様の行為を続けた場合の結果を伴う警告。一定期間、関係者とのやり取りや、行動規範を施行する人への直接連絡を含むあらゆる非要請の交流を禁止します。コミュニティ内や外部チャンネル（SNS 等）での交流も対象です。違反が続く場合は一時的または永久的な利用停止に繋がる可能性があります。

### 3. 一時的な利用停止

**コミュニティへの影響**: コミュニティ基準の重大な違反、または継続的な不適切行為。

**結果**: 指定期間、コミュニティとの公開・非公開の交流を禁止します。この期間中は、関係者や行動規範施行者への非要請の連絡も認められません。違反した場合は永久的な利用停止となることがあります。

### 4. 永久的な利用停止

**コミュニティへの影響**: コミュニティ基準の反復的な違反、特定個人への嫌がらせ、特定集団への敵対や侮辱を含む行為。

**結果**: コミュニティ内での公的な交流の永久的な禁止。
